# 🚀 Smart Interview Practice Platform

A Flask-based web application designed to help students prepare for coding interviews by practicing programming questions, tracking progress, and improving problem-solving skills.

## ✨ Features

- 🔐 User Authentication (Login & Registration)
- 💻 Coding Problem Practice
- ❤️ Favorite Problems
- 📈 Progress Tracking
- 📊 Statistics Dashboard
- 📝 Practice Sheet Generator
- 👨‍💼 Admin Dashboard
- 🔍 Problem Details Page

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- SQLAlchemy

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Anurajpal6666/Smart-Interview-Practice-Platform-.git
```

Go to the project folder:

```bash
cd Smart-Interview-Practice-Platform-
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
